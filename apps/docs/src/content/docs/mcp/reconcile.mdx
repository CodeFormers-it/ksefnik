Narzędzie `reconcile` uruchamia silnik uzgadniania, który dopasowuje zsynchronizowane faktury KSeF do zaimportowanych transakcji bankowych.

## Parametry wejściowe

Brak. Narzędzie operuje na danych już załadowanych do kontekstu przez `sync-invoices` i `import-bank`.

<Aside type="caution">
Przed wywołaniem `reconcile` musisz wykonać `sync-invoices` i `import-bank`. Bez danych w kontekście narzędzie zwróci pusty raport.
</Aside>

## Format odpowiedzi

```ts
{
  summary: {
    matched: number
    unmatchedInvoices: number
    unmatchedTransactions: number
    averageConfidence: number
  }
  matches: Match[]
  unmatchedInvoices: Invoice[]
  unmatchedTransactions: BankTransaction[]
  reportId: string
}
```

## Przykład wywołania

```json
{
  "name": "reconcile",
  "arguments": {}
}
```

## Przykład odpowiedzi

## Kiedy używać

- Po zsynchronizowaniu faktur i zaimportowaniu wyciągu
- Gdy użytkownik prosi "uzgodnij faktury z przelewami"
- Jako krok w procesie miesięcznego zamknięcia

<Aside type="tip">
Pole `reportId` potrzebne jest do wywołania `get-unmatched` i `confirm-match`. Zachowaj je do dalszego przetwarzania.
</Aside>

## Powiązane narzędzia

- `get-unmatched` -- pobierz szczegóły nieuzgodnionych pozycji
- `confirm-match` -- potwierdź lub odrzuć dopasowanie
- `sync-invoices` -- pobierz faktury (wymagane przed reconcile)
- `import-bank` -- zaimportuj wyciąg (wymagane przed reconcile)
