Narzędzie `sync-invoices` pobiera faktury z Krajowego Systemu e-Faktur za wskazany zakres dat i zapisuje je w lokalnym kontekście do dalszego przetwarzania.

## Parametry wejściowe

| Parametr | Typ | Wymagany | Opis |
|----------|-----|----------|------|
| `dateFrom` | `string` | Tak | Data początkowa (YYYY-MM-DD) |
| `dateTo` | `string` | Tak | Data końcowa (YYYY-MM-DD) |
| `nip` | `string` | Nie | NIP podmiotu (domyślnie z konfiguracji) |

## Format odpowiedzi

```ts
{
  invoices: Invoice[]
  count: number
}
```

## Przykład wywołania

```json
{
  "name": "sync-invoices",
  "arguments": {
    "dateFrom": "2026-03-01",
    "dateTo": "2026-03-31"
  }
}
```

## Przykład odpowiedzi

```json
{
  "invoices": [
    {
      "ksefReference": "KSeF-1234567890-20260310-ABC123",

      "grossAmount": 123000,
      "issueDate": "2026-03-10"

      "buyerNip": "5555555555",

```

## Kiedy używać

- Na początku procesu uzgadniania -- pobierz faktury przed importem wyciągu
- Gdy AI potrzebuje aktualnych danych z KSeF
- Okresowa synchronizacja (np. "pobierz faktury z ostatniego miesiąca")

<Aside type="tip">
Kwoty są w groszach (100000 = 1000,00 PLN). AI powinien przeliczać je na złote w odpowiedziach dla użytkownika.
</Aside>

## Powiązane narzędzia

- `query-invoices` -- wyszukiwanie wśród już zsynchronizowanych faktur
- `reconcile` -- uzgodnienie pobranych faktur z przelewami
