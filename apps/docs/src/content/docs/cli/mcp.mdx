Komenda `mcp` uruchamia serwer Model Context Protocol, który pozwala asystentom AI (Claude, Cursor) korzystać z ksefnik jako narzędzia.

## Użycie

```bash
ksefnik mcp [--stdio]
```

## Parametry

| Parametr | Wymagany | Opis |
|----------|----------|------|
| `--stdio` | Nie | Użyj transportu stdio (domyślnie) |

## Działanie

Serwer MCP nasłuchuje na stdin i odpowiada na stdout, zgodnie ze specyfikacją MCP. Proces nie kończy się sam -- działa do momentu zamknięcia strumienia wejściowego.

Udostępniane narzędzia MCP:
- `sync-invoices` -- synchronizacja faktur z KSeF
- `import-bank` -- import wyciągu bankowego
- `reconcile` -- uzgadnianie faktur z przelewami
- `get-unmatched` -- pobranie nieuzgodnionych pozycji
- `query-invoices` -- wyszukiwanie faktur
- `send-invoice` -- wysyłka faktury
- `validate-invoice` -- walidacja faktury
- `confirm-match` -- potwierdzenie dopasowania
